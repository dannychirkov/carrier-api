# 1.1 Особистий кабінет API 2.0

**a.** API (інтерфейс програмування додатків) - це набір інструментів для автоматизації роботи з компанією « Нова Пошта ». Функціональність API дозволяє швидко інтегрувати логістичні процеси в будь-який бізнес і є єдиною точкою входу для всіх клієнтів і сервісів.  
**b.** <https://api.novaposhta.ua/> \- це портал для розробників який присвячений інтеграції API «Нова Пошта», на ньому представлена актуальна розширена документація, приклади та інструменти розробника. Реалізована можливість спробувати запити на тестовому середовищі.   
**c.** API key - для початку роботи з функціоналом АРІ компанії «Нова Пошта» необхідно згенерувати ключ API та використовувати при формуванні запиту, що потребує використання API-ключа. Ключ можна згенерувати з меню особистого кабінету користувача за посиланням <https://my.novaposhta.ua/settings/index#apikeys>  
**d.** Формат запиту - обмін даними за допомогою API здійснюється по протоколу HTTP(S) з використанням способу передачі даних POST/GET на точку входу в залежності від типу запиту XML або JSON
