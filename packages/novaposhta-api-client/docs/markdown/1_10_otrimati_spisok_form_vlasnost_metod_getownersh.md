# 1.11 Отримати список форм власності (Метод «getOwnershipFormsList»)


Метод **«getOwnershipFormsList»** працює в моделі **"Common"** цей метод дозволяє завантаження довідника списку форм власності. Необхідно зберігати копію довідників на стороні клієнта та підтримувати її в актуальному стані. Рекомендується проводити оновлення довідників раз на місяць.   
Доступність: Потребує використання API-ключа.

### XML
    
    
    <?xml version="1.0" encoding="UTF-8"?>
    <file>
    <apiKey>Ваш ключ АРІ 2.0</apiKey>
    <calledMethod>getOwnershipFormsList</calledMethod>
    <methodProperties>
    </methodProperties>
    <modelName>Common</modelName>
    </file>
    

### JSON
    
    
    {
    "apiKey": "Ваш ключ АРІ 2.0",
    "modelName": "Common",
    "calledMethod": "getOwnershipFormsList",
    "methodProperties": {}
    }
    

### Опис відповіді на запити

Параметри | Тип | Опис  
---|---|---  
Description | string[50] | форма власності (скорочено)  
Ref | int[36] | ідентифікатор форми властності  
FullName | string[] | Форма власності (повна назва)
