# 1.7 Отримати список видів та форм платників («getTypesOfPayer»)

Метод **«getTypesOfPayer»** працює в моделі **"Common"**, цей метод завантажує список видів платників послуги доставки: Відправник, Отримувач, Третя особа. Вид платника «Третя особа» можливо замовити тільки після укладення договору з компанією «Нова Пошта». Необхідно зберігати копію довідників на стороні клієнта та підтримувати її в актуальному стані. Рекомендується проводити оновлення довідників раз на місяць.   
Доступність: Потребує використання API-ключа.

### XML
    
    
    <?xml version="1.0" encoding="UTF-8"?>
    <file>
    <apiKey>Ваш ключ АРІ 2.0</apiKey>
    <calledMethod>getTypesOfPayers</calledMethod>
    <methodProperties />
    <modelName>Common</modelName>
    </file>
    

### JSON
    
    
    {
      "modelName": "Common",
      "calledMethod": "getTypesOfPayers",
      "apiKey": "ваш ключ АРІ 2.0",
      "methodProperties": {}
    }
    

### Опис відповіді на запити

Параметри | Тип | Опис  
---|---|---  
Description | string[50] | Тип платника (Відправник/Одержувач/Третя особа)  
Ref | int[36] | Ідентифікатор типу платника в АРІ (Sender/Recipient/ThirdPerson)  
  
* * *

Метод **«getPaymentForm»** працює в моделі **"Common"**, цей метод завантажує список форм оплати: готівковий розрахунок, безготівковий розрахунок.Оплачувати послугу доставки за допомогою безготівкового розрахунку можуть Клієнти, які заключили договір з компанією «Нова Пошта». Необхідно зберігати копію довідників на стороні клієнта та підтримувати її в актуальному стані. Рекомендується проводити оновлення довідників раз на місяць.   
Доступність: Потребує використання API-ключа.

### XML
    
    
    <?xml version="1.0" encoding="UTF-8"?>
    <file>
    <apiKey>Ваш ключ АРІ 2.0</apiKey>
    <calledMethod>getPaymentForm</calledMethod>
    <methodProperties />
    <modelName>Common</modelName>
    </file>
    

### JSON
    
    
    {
      "modelName": "Common",
      "calledMethod": "getPaymentForm",
      "apiKey": "ваш ключ АРІ 2.0",
      "methodProperties": {}
    }
    

### Опис відповіді на запити

Параметри | Тип | Опис  
---|---|---  
Description | string[50] | Тип платника (Безготівковий/Готівка)  
Ref | int[36] | Ідентифікатор типу платника в АРІ (NonCash/Cash)
